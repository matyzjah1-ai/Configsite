Github VM Script Generator

Generate automated workflow scripts for self-hosted cloud environments via GitHub Actions.

Tailscale

Setup Process:

  1. Get your API / Auth Token from the console at Tailscale.
  2. Make sure the key settings have enabled:
    • Reusable
    • Ephemeral
  3. Go to your GitHub Repository → Settings → Secrets and variables → Actions.
  4. Create a New repository secret named TAILSCALE_AUTH_KEY and paste your key value there.

Lets go! Your Script is made

Now go to your repository, click Actions, select "Set up a workflow yourself", paste this script, then trigger the workflow under Actions.

Remember the VM Lasts 6 Hours maximum per run
